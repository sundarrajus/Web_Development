## 📌 About This Repository

This repository contains all notes, code, and projects from a full web development course. It covers everything from the fundamentals of HTML to dynamic JavaScript and responsive Bootstrap design, culminating in real-world projects that bring all the skills together.

Whether you're a complete beginner or looking to solidify your foundations, this course takes you through each layer of front-end web development step by step.

## 🧭 What You'll Learn

### 🟠 HTML — The Structure
Learn to build the skeleton of any webpage using semantic, accessible markup.
- Document structure, headings, paragraphs, links, images
- Tables, forms, inputs, and media elements
- Semantic HTML5 tags for better accessibility and SEO

### 🔵 CSS — The Style
Master how to make webpages look beautiful and professional.
- Selectors, the Box Model, colors, fonts, and backgrounds
- Flexbox and Grid for powerful, modern layouts
- Transitions, animations, and responsive design with media queries

### 🟣 Bootstrap — The Framework
Speed up development using the world's most popular CSS framework.
- Grid system, containers, and responsive breakpoints
- Components: Navbar, Cards, Modals, Carousel, Forms, Buttons
- Bootstrap utility classes and icons

### 🟡 JavaScript — The Interactivity
Add life to your pages with dynamic, client-side programming.
- Variables, operators, conditionals, and loops
- Functions, arrays, objects, and the DOM
- Events, the Fetch API, async/await, and Local Storage

### 🟢 Projects — The Practice
Apply everything you've learned by building real, deployable projects.
- Personal Portfolio Website
- Responsive Landing Page
- Bootstrap Business Page
- Calculator & To-Do List App
- Weather App using a live API
- Quiz App & Form Validation

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge)
- A code editor — [VS Code](https://code.visualstudio.com/) is recommended
- [Live Server extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) for VS Code (optional but helpful)

### How to Use This Repo

1. **Clone the repository**
   ```bash
   git clone https://github.com/sundarrajus/Web_Development.git
   ```

2. **Navigate to the course folder**
   ```bash
   cd Web_Development/Web_Development_Full_Course
   ```

3. **Open any HTML file** in your browser or with Live Server to see it in action.

4. **Follow the sections in order:** HTML → CSS → Bootstrap → JavaScript → Projects

## 📂 Folder Details

### `HTML/`
Covers all fundamental and advanced HTML topics with practice files for each concept. Each file is self-contained and demonstrates a specific tag or concept.

Complete, fully working mini-projects combining HTML, CSS, Bootstrap, and JavaScript. Each project folder is standalone and ready to open in the browser.

## 💡 Tips for Learners

- **Don't skip HTML and CSS.** They are the foundation — JavaScript makes no sense without them.
- **Type every line of code.** Don't just read or copy-paste; muscle memory matters.
- **Open DevTools** (F12) to inspect elements, debug JS, and see styles live.
- **Experiment freely.** Break things intentionally — it's the fastest way to learn.
- **Build the projects yourself** before looking at the solution code.
