## Overview

A complete, structured **Web Development Full Course** repository covering the core technologies of the web - **HTML**, **CSS**, **JavaScript**, and **Bootstrap 5** — across 5 phases. From building your first webpage to DOM manipulation, responsive design with Bootstrap's grid system, and interactive JavaScript projects.

This repository is designed for learners who want a clean, well-organized reference with runnable code examples, commented explanations, real-world mini projects, and a structured study roadmap.

---

## What's Covered

### Phase 1 — HTML Foundations
- HTML document structure, tags, attributes
- Headings, paragraphs, links, images
- Lists (ordered, unordered, nested)
- Tables — rows, columns, colspan, rowspan
- Forms — input types, labels, buttons, validation attributes
- Semantic HTML5 — header, nav, main, section, article, footer

### Phase 2 — CSS Fundamentals
- Selectors — element, class, ID, pseudo-class, pseudo-element
- Box model — margin, border, padding, content
- Colors, backgrounds, gradients
- Typography — fonts, sizes, weights, Google Fonts
- Display, position, float, z-index
- Flexbox — container and item properties
- CSS Grid — rows, columns, areas, responsive layouts
- Transitions and animations

### Phase 3 — Bootstrap 5
- CDN setup — linking Bootstrap CSS and JS
- Bootstrap grid system — container, row, col, breakpoints (sm, md, lg, xl)
- Utility classes — spacing (m-*, p-*), display, text alignment, colors
- Typography & color utilities
- Components — Navbar (responsive + toggler), Cards, Buttons, Badges, Alerts
- Forms with Bootstrap — form-control, form-label, input-group, validation styles
- Bootstrap Flexbox utilities — d-flex, justify-content-*, align-items-*
- Modal, Accordion, Carousel
- Responsive Bootstrap project — building a full page using Bootstrap only

### Phase 4 — JavaScript Core
- Variables — var, let, const
- Data types — string, number, boolean, null, undefined, object
- Operators, conditionals, loops
- Functions — declarations, expressions, arrow functions
- Arrays — methods (map, filter, reduce, forEach, find)
- Objects — creation, access, methods, destructuring
- DOM manipulation — querySelector, getElementById, innerHTML, classList
- Events — addEventListener, click, input, submit, keydown
- Form validation with JavaScript

### Phase 5 — Projects & Advanced Concepts
- Responsive Design — media queries, mobile-first approach
- CSS Variables (Custom Properties)
- JavaScript ES6+ features — spread, rest, template literals, optional chaining
- Local Storage — save and retrieve data
- Fetch API basics — working with JSON data
- Mini Projects — interactive UI components and apps

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge)
- A code editor (VS Code recommended)
- Git (optional, for cloning)

### Run Locally

1. **Clone the repository**
   ```
   git clone https://github.com/sundarrajus/Web_Development.git
   cd Web_Development
   ```

2. **Navigate to the course folder**
   ```
   cd Web_Development_Full_Course
   ```

3. **Open any lesson in your browser**
   ```
   open 01_html_basics.html
   ```
   Or simply double-click any `.html` file — no server or build step required.

4. **Recommended: Use Live Server in VS Code**
   Install the Live Server extension and click "Go Live" for auto-reload on save.
