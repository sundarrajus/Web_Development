Bootstrap Basics

Bootstrap is a free CSS framework. It gives you ready-made classes so you don't have to write CSS from scratch!

1. Utility Classes — Spacing (m, p)

Bootstrap uses m for margin and p for padding. Numbers go from 0 to 5.

p-3 = padding 3, mb-2 = margin-bottom 2
p-1 ms-4 = small padding, margin-start 4
