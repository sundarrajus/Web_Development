Bootstrap Grid System

Bootstrap divides every row into 12 equal columns. You combine columns to create layouts.

1. Equal Columns — col

Just use col — Bootstrap splits space equally.

2. Specific Widths — col-6, col-4, col-3...

Numbers must add up to 12. col-12 = full width, col-6 = half, col-4 = one-third, col-3 = one-quarter.

3. Responsive Grid — Breakpoints (sm, md, lg)

Breakpoints let columns change at different screen sizes:
col-sm-6 → 2 columns on small screens and up
col-md-4 → 3 columns on medium screens and up
col-lg-3 → 4 columns on large screens and up

4. Offset — Push Columns

offset-md-3 skips 3 columns, then your column starts.
