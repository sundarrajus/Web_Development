CSS Box Model

1. The CSS Box Model — Visual Diagram

Every HTML element is a rectangular box. It has 4 layers:

Content → Padding → Border → Margin

CONTENT

2. Margin — Space OUTSIDE the border

margin: 30px auto; — 30px top/bottom, auto centers horizontally

Shorthand: margin: top right bottom left

margin: 30px auto

3. Padding — Space INSIDE the border

Padding pushes content away from the border. The background color fills the padding area.

padding: 40px

4. Border — The outline around the element

Border can have different width, style, and color for each side.

Different border on each side!

6. box-sizing: border-box

By default, padding and border add to the element's width. With box-sizing: border-box, the total size stays as you set — padding and border are included inside.

That's why we write * { box-sizing: border-box; } at the top of every CSS file!
