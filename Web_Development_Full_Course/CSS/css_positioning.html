CSS Positioning

1. position: static — Default (Normal Flow)

Every element is static by default. Elements appear one after another — no special positioning.

Box 1 — static
Box 2 — static
Box 3 — static

2. position: relative — Moves from its normal spot

position: relative moves the element from WHERE IT WOULD NORMALLY BE using top, left, right, bottom.

I moved 20px down, 40px right from my normal position

3. position: absolute — Positions inside the nearest relative parent

The child with position: absolute is placed relative to its nearest positioned parent (parent with position: relative/absolute/fixed).

Key rule: Always give the parent position: relative first!

Top-Right
Center (transform trick)
Bottom-Right

4. position: fixed — Sticks to the screen

position: fixed keeps the element in the same spot even when you scroll. Common use: sticky headers, floating buttons.

Example: position: fixed; top: 0; left: 0; pins to top-left of the screen.
