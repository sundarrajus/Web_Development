CSS Selectors & Pseudo-classes

1. Element Selector — p { }

Targets all elements of that type. p { color: darkblue; }

This paragraph is styled by the element selector.

This one too — every <p> inside .demo1 is affected.

2. Class Selector — .classname { }

Targets elements with that class. Multiple elements can share a class.

Normal paragraph — no class

This has class="highlight" → yellow background

This also has it!

3. ID Selector — #id { }

Targets ONE unique element. IDs must be unique on the page.

This has id="special" → red, bold, large

Normal paragraph

4. Grouping & Descendant Selectors

h3, p { } targets both. .parent .child { } targets child inside parent.

This h3 is orange (grouped)

This p is also orange (grouped)

Parent div
I am .child inside .parent — green!

5. :hover Pseudo-class

Applies styles when the mouse hovers over an element.

Hover over me!
  

6. :focus Pseudo-class

Applies styles when the element is focused (clicked into). Very useful for form inputs.

7. :first-child, :last-child, :nth-child()

Target list items by their position.

  • First item — green (first-child)
  • Second item — blue (nth-child(2))
  • Third item — normal
  • Fourth item — normal
  • Fifth item — red (last-child)

8. ::before & ::after Pseudo-elements

Add content before or after an element without touching the HTML.

This text has icons added by CSS!
