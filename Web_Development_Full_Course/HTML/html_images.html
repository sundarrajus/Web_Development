HTML Images — Complete Guide

1. Basic img Tag

Syntax: <img src="path/to/image.jpg" alt="description">

The src attribute gives the path to the image. The alt text shows if image fails to load.

2. Rounded Corners — border-radius

Use border-radius in CSS to round the corners of an image.

3. Circle Image — border-radius: 50%

Setting border-radius: 50% makes any image a circle. Use equal width and height!

4. Image with Border

Add a border property in CSS to give the image an outline.

5. Responsive Image — width: 100%

Set width: 100% so the image always fits its container — this is responsive design!

6. object-fit: cover vs contain

object-fit: cover — fills the box, may crop the image (most common for cards/thumbnails)

object-fit: contain — fits the entire image inside the box, may leave empty space

7. Image Inside a Fixed Container (like a card)

Wrap the image in a <div> with fixed size and overflow: hidden. This is very common in card layouts.

8. Image Grid (using flexbox)

Combine images with display: flex to create a photo gallery grid.
